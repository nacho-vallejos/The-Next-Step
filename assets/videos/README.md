# Videos

Coloca aquí el video de fondo:

## Archivo requerido:
- **background.mp4** - Video de fondo lunar para la página

## Recomendaciones:
- Resolución: 1920x1080 o superior
- Duración: 10-30 segundos (se reproducirá en loop)
- Formato: MP4 (H.264)
- Tamaño: Preferentemente menor a 10MB para mejor rendimiento
- Contenido sugerido: Superficie lunar, primera huella en la Luna, o animación espacial

## Fuentes sugeridas para videos:
- Pexels: https://www.pexels.com/search/videos/moon/
- Pixabay: https://pixabay.com/videos/search/lunar/
- Coverr: https://coverr.co/
- NASA: https://images.nasa.gov/

## Nota:
El video actual está configurado con:
- `autoplay` - Se reproduce automáticamente
- `muted` - Sin sonido
- `loop` - Reproducción continua
- `playsinline` - Compatible con dispositivos móviles
- Opacidad del 12% para no interferir con el contenido
